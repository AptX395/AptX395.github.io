小米手机刷国际版欧版 MIUI 的详细教程

发表于 2019-05-27 更新于 2019-06-07 分类于 Tutorial
本文主要介绍了小米手机刷欧版（最纯净） MIUI 的详细教程。

Perface

为什么要刷欧版 MIUI 而不是刷普通的国际版就行了？

普通的国际版 MIUI 虽然没有国内版 MIUI 广告多，但是仍有 AnalyticsCore、MSA-Global 等广告后台，而且系统预装了谷歌全家桶，比较臃肿。欧版 MIUI 是得到欧盟分销商 ABC Data 认可的、真正纯净的，保留了基本的谷歌服务框架，但没有预装谷歌全家桶，十分清爽；包括其中的小米音乐、小米视频等系统软件都经过了去广告处理。

Prerequisite

  1. 解锁你的小米手机

    到这里申请解锁。根据申请成功后的提示，下载解锁工具。下载后解压。

  2. 下载欧版 MIUI ROM 包

    到这里下载与你的手机机型对应的 ROM 包。下载后不需要解压。

  3. 下载 Andriod Debug Bridge (ADB) SDK

    到这里下载与你的电脑操作系统对应的 ADB SDK 。下载后解压。

  4. 下载第三方 Recovery TWRP

    到这里下载与你的手机机型对应的 TWRP 镜像文件 。

解锁手机

  1. 首先在手机处于开机状态下打开解压好的解锁工具的文件夹中的 miflash_unlock.exe ，然后用 USB 连接手机，第一次连接会安装必要的驱动。
  2. 安装好驱动后，断开手机的连接，将手机关机。
  3. 手机关机后，同时按住电源键和音量下键，几秒后会进入 fash boot 模式。
  4. 进入 fast boot 模式后再次将手机连接电脑，解锁工具会自动识别手机，根据提示进行解锁即可。

刷入 ROM

完成上面的步骤后，手机会进入 TWRP 的界面，可以设置显示的语言。

  1. 进行双清

    点 Wipe ，勾选 system, cache, data 然后滑动进行清除。

  2. 刷入 ROM

    回到主界面，点 Install 选择 ROM 包进行刷机，等待完成，完成后重启手机就完成了整个刷机过程。

    PS: ROM 包可以提前 Copy 到手机的存储中，也可以在 TWRP 工具主界面点 Mount 然后点 Enable MTP ，这样你就可以从电脑向手机 Copy ROM 包了。
